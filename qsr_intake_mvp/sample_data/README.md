# Sample Data

All files in this directory are synthetic demo data for local development and portfolio review.

They are included to show:

- source-shaped raw payloads and exports
- normalization behavior across POS, labor, inventory, and Aloha-style scenarios
- review-required and unresolved cases in the metadata path

They are not production datasets, partner datasets, or real personnel records. Any names, identifiers, stores, wages, or tips shown here are fabricated for demonstration purposes.
